## Visual Design System

### Colors

**Primary:**

- **Blue:** #2563EB (Notion-style blue)
- **Purple:** #7C3AED (accent, gradients)

**Neutrals:**

- **Dark:** #1F2937 (headings, dark sections)
- **Gray:** #6B7280 (body text)
- **Light Gray:** #F3F4F6 (section backgrounds)
- **White:** #FFFFFF

**Semantic:**

- **Success Green:** #10B981
- **Warning Orange:** #F59E0B
- **Error Red:** #EF4444

### Typography

**Headings:**

- Font: Inter, SF Pro, or system-ui
- H1: 48px (mobile: 32px), bold
- H2: 36px (mobile: 28px), bold
- H3: 24px (mobile: 20px), semi-bold

**Body:**

- Font: Inter or system-ui
- Size: 18px (mobile: 16px)
- Line height: 1.6
- Color: #6B7280

**Buttons:**

- Font: Inter, semi-bold
- Size: 16px
- Padding: 12px 32px
- Border radius: 8px
- Primary: Blue background, white text
- Secondary: White background, blue border, blue text

### Icons

**Library:** Lucide Icons or Heroicons (consistent with app)

**Sizes:**

- Feature icons: 48px
- Step icons: 64px
- Inline icons: 20px

**Style:** Outline (not filled), 2px stroke

### Spacing

**Sections:**

- Vertical padding: 80px (desktop), 48px (mobile)
- Max width: 1200px
- Side padding: 24px (mobile), 48px (desktop)

**Elements:**

- Headline to subheadline: 16px
- Subheadline to content: 32px
- Between cards/features: 24px

## Mobile Optimization

### Critical Considerations

**97% of users are mobile**[[1]](https://www.notion.so/MVP-Plan-ffb2629a0e624b9196a23d4a4eff2855?pvs=21) - mobile MUST be perfect

**Mobile-Specific Adjustments:**

1. **Hero:**
   - Stack vertically (headline â†’ subheadline â†’ button)
   - Make button full-width on mobile
   - Reduce headline size (48px â†’ 32px)
2. **Problem Statement:**
   - Stack 3 columns vertically
   - Add more spacing between cards
3. **How It Works:**
   - Vertical flow (not horizontal)
   - Larger icons (easier to see)
4. **Features:**
   - One column only
   - Larger text (18px minimum)
5. **Pricing:**
   - Stack cards vertically
   - Highlight "Starter" with border/shadow
6. **Forms:**
   - Full-screen modal (not popup)
   - Large input fields (48px height minimum)
   - Large buttons (48px height minimum)

**Touch Targets:**

All buttons and links: minimum 44x44px
